开发参与注意事项：

1.版本号规则：
	1.1 版号号组成规则：
		1.1.1 主版本.次版本.内部版本号.修订号
		1.1.2 有极大更新，或者版本无法向下兼容时，主版本号加+1，有较大更新时，次版本号+1，BUG修复后发布时，内部版本号+1，每次代码更新，修订号+1
	1.2 程序集版本号：从2015/11/24起，修订号不再使用增长的数字，统一使用6位日期加二个数字，如：1.3.1.15112401，如果当日进行多次更新，则后二位数字进行增长
	1.3 文件版本号：同 程序集版本号 但是 省略修订号

2.引用/依赖规则：
	项目必须只依赖最小的dll(即只引用System),以保证项目的可移植性。

3.参与规则：
	如果您想对jntemplate进行代码贡献，请在开源主页（https://github.com/jiniannet）上fork  develope分支，提交您的代码后，向我们进行 pull request即可！
